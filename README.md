# Pingshell 

***Pingshell is a small program that allows you to send commands to a target host using only ICMP pings. This functionality is complete, but many improvements can still be made. Currently the communication is already obfuscated into binary and transmitted over empty packets, but in theory I could have the message encrypted before sending and decrypted on the other side. I could also encrypt the data from the host going back to the C2 infrastructure. I also plan to make a way to check for dropped packets and replace the data. I have also created its counterpart, PingSmell. PingSmell is a daemon that can constantly check for binary data smuggled between 2 addresses.***
